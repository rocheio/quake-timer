# Quake Timer

Quake cooldown timer app for the game Quake Champions.

Run the app:

```sh
go run main.go
```

Turn your volume up so you can hear the dings.

Press the registered hotkey, even when terminal is not focused.

`Alt + 1`

You should hear a 'ding'
